txtool is made to help you for easly pentesting in termux,  
build on termux and only available for termux.  

# installation steps: 
* require python version 2.x  

```
$ git clone https://github.com/kuburan/txtool.git  
$ cd txtool  
$ apt install python2  
$ ./install.py 
$ txtool
```
